Multilingual Analysis of Names

The website Listophile has a page of Names by Nationality. I have downloaded the names, and the nationality names, into two files, formatted using TOML (TOML: Tom's Obvious Minimal Language):

  • names-listophile-desc.toml
  • names-listophile.toml

Read in the files, and investigate the names:

  1. How many nationalities are there?
  2. How many names are there for each nationality?
    Give examples
  3. How many names are ambiguous between genders for each language:
    Give examples
  4. Learn a gender classifier for each nationality:
    • What are the most useful features for each language?
    • What are the most useful features for all languages?
    • Are names in some languages more sharply distinguished than others?
  5. Two languages (Korean and Chinese) have the names in multiple scripts
    • Train classifiers for each script
  6. The names are given with descriptors: Francis "Free man"
    • Are the descriptors consistent across languages?
    • Can you predict gender from the descriptors?
      Even if you take out 'man', 'woman'?
