LAC: Language and the Computer

Project 1: Japanese names

This project constitutes 30% of your final grade for LAC. Please work on the final program and report individually.

You will be given some name data from the website Heisei Namae Jiten and use the jamdict module for accessing Jim Breen's kanjidic.

There is a some scaffolding to help you get started: Here is the code in collab: 2024-project1.ipynb, and here is the data you needs names.tsv. Please feel free to extend this or start from scratch as you prefer.

names.tsv has the following columns: 

type    year    rank    name     frequency
boy     1       1       翔太     4639

Where type is one of boy/girl/name for boy's name, girl's name, full name. Year is the year of the Heisei Era (year 1 is from 8 January 1989, year 21 is to 30 April 2019).

Minimal Requirements

  1. Measure the average number of stroke counts
    1. For full names
    2. For first names
    3. For last names
    Make sure you do something for hiragana
  2. Are first names or last names more complicated (i.e. have more strokes)
    Is this difference significant?

The data is sorted by year, and we know the names of boys and girls.

Stretch Goals

  1. Look at differences between boys and girls.
  2. Look at differences between years (don't forget to normalize for frequency: there may be different totals for different years).
  3. Are some stroke counts more popular than others?
  4. Do people with complicated surnames tend to have simpler first names or not?
