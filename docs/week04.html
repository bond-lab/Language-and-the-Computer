Lecture notes

  • Read Python Crash Course Chapter 6 and 7 (the last time we do two chapters)
  • A dictionary is a collection of key-value pairs which is indexed and changeable. Keys must be unique. (from Python 3.7 it is ordered)
    vocab = dict()
    myvocab = {"kore":"これ", "sore":"それ", "are":"あれ" }
    vocab['kore'] = 'これ'
  • Set is a collection which is unordered (and thus unindexed) and changeable. No duplicate members.
    vocab = set()
    vocab.add('word') #not append
    myvocab = {"kore", "sore", "are"}
  • while loops continue until the condition is not true
    be careful not to loops forever
  • the input() function in Python reads a line of text input from the user and returns it as a string.
    name = input("Enter your name: ")
  • Cheat Sheet: if/while, Cheat Sheet: if/while BW, Cheat Sheet dict., Cheat Sheet dict. BW

Summary

  • how while loops can be used with dictionaries
